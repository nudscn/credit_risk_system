# 需求冻结 V1（中间层子项目）

## 一、项目边界
1. 本阶段仅开发“基础数据中间层”。
2. 暂不开发最前端数据获取（OCR/PDF识别入口）。
3. 暂不开发后端信用风险分析与评级结论引擎。

## 二、目标交付物
产出单一 Excel 文件：`***基础数据.xlsx`（多 sheet）。

### 必含内容
1. 标准财务报表模板（自建）。
2. 自动填充后的报表（按来源规则填值）。
3. 差异/缺失清单。
4. 报表层面勾稽校验结果。
5. 财务比率规则与结果。
6. 各科目明细表（每个科目一个 sheet，参考既有底稿风格）。

## 三、模板标准与口径
1. 模板标准：企业会计准则（一般企业）。
2. 口径：统一采用“合并报表”。
3. 报表范围：
   - 资产负债表
   - 利润表
   - 现金流量表
   - 所有者权益变动表（一级颗粒度）
   - 现金流补充资料

## 四、期间与单位规则
1. 不考虑中报，仅年报。
2. 期间规则：以最新期间向前倒推三年（动态）。
3. 单位统一：万元，保留 2 位小数。
4. 负数显示：括号 + 红色字体。

## 五、填充策略
1. 先生成标准模板，再按来源搜索并填充。
2. 来源优先级：结构化 Excel 底稿 > 审计报告 > 评级报告。
3. 若第一优先来源数据通过勾稽校验，则认定可用。
4. 未填到值统一标记“待补充”。

## 六、勾稽规则（V1）
### A层 恒等式（必须）
1. 资产总计 = 负债合计 + 所有者权益合计。
2. 期末现金及现金等价物余额 = 期初现金及现金等价物余额 + 现金及现金等价物净增加额。
3. 所有者权益（年末）= 所有者权益（年初） + 本年净利润（一级口径；差异需提示）。

### B层 小计合计（必须）
1. 流动资产明细求和 = 流动资产合计。
2. 非流动资产明细求和 = 非流动资产合计。
3. 流动资产合计 + 非流动资产合计 = 资产总计。
4. 流动负债明细求和 = 流动负债合计。
5. 非流动负债明细求和 = 非流动负债合计。
6. 流动负债合计 + 非流动负债合计 = 负债合计。
7. 利润表关键小计关系（营业利润/利润总额/净利润）。
8. 现金流量表各活动流入流出小计与净额关系。

### C层 期间滚动（高优先）
1. 上期年末 = 下期年初（同口径同科目）。

## 七、财务比率（V1）
1. 体系：杜邦分析 + 偿债 + 运营 + 盈利 + 发展。
2. 余额口径：优先平均余额；无法获取时使用期末余额。

## 八、结果判定与提示
1. 勾稽 sheet 输出形式：规则描述 + 结果（是/否） + 差异值。
2. 任一关键规则不通过，标记“需复核”。
3. 缺失与异常自动进入差异/缺失清单。
